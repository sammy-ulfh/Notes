## ¿Que es HTML?

HTML es un lenguaje de marcado de hipertexto, el cual nos sirve para realizar un enmaquetado mediante etiquetas de los elementos que vayamos a definir dentro de una pagina web. Este enmaquetado es interpretado por un navegador.

En una pagina web HTML es la estructura primordial al permitir enmaquetar texto, imagenes, botones, tablas, etc. HTML no tiene que ver directamente con la parte visual de una pagina web, para ello se utiliza un lenguaje de cascada el cual nos permite aplicar "estilos" y mejorar visualmente nuestra estructura realizada con HTML.

Por ende, esto podremos verlo como que con HTML etiquetamos el contenido de nuestra pagina web y con CSS aplicamos los estilos para mejorar nuestro apartado visual.

Para poder trabajar con HTML es necesario unicamente un navegador (cualquiera de los mas usados actualmente) que pueda interpretar y mostrarnos nuestro enmaquetado de contenido.

Es importante considerar que los archivos html, tendran la extension ".html".

## Contruir mi primer archivo

Para tu primera pagina tendras que crear un archivo index.html, este se edita como si de texto se tratase. Se puede editar con el bloc de notas, pero en este caso iremos mas alla. Utilizaremos un editor de codigo, ya sea [VSCode]([Visual Studio Code - The open source AI code editor](https://code.visualstudio.com/)) o [SublimeText]([Download - Sublime Text](https://www.sublimetext.com/download)).

El archivo podremos abrirlo con doble click, ya que asi, directamente nos lo abriria el navegador. En caso de que no, podriamos dar click derecho y seleccionar "abrir con" e indicar un navegador. 
Cuando estemos realizando cambios en nuestra pagina veremos como estos cambios después de guardar el contenido dle archivo no lo veremos reflejado directamente, es por ende que tendremos que darle al boton de recargar pagina en el navegador, para ir viendo los cambios aplicados.
