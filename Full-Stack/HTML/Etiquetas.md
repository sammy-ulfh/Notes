## ¿Que es una etiqueta?

Una etiqueta en html es un tipo de envoltura que nos servira para envolver de forma estructurada el contenido de nuestra pagina web.

Cada etiqueta en html es de ayuda para mostrar cosas de forma distinta. La estructura de una etiqueta contiene como apertura y cierra la misma etiqueta, pero la de cierre tendra un slash.

```HTML
<etiqueta></etiqueta>
```

En HTML existen una gran variedad de etiquetas para mostrar distintas cosas en una pagina web. Existen etiquetas para texto, parrafos, videos, sonido, imagenes, encabezados, tablas, negritas, etc. Cada etiqueta cumple una funcion distinta y se iran observando a lo largo del curso.

## Estructura de una etiqueta

Una etiqueta en HTML podra tener una serie de atributos para indicar cierto comportamiento de la etiqueta.

```HTML
<etiqueta atributo="valor">Datos o texto</etiqueta>
```

Por ejemplo, si utilizamos como atributo style, podremos indicar una serie de instrucciones con CSS para modificar la parte visual de nuestro elemento con el lenguaje de cascada.

## Primera etiqueta

Etiqueta __\<strong>__ => En esta etiqueta podremos envolver un texto que quieramos mostrar y este en nuestra pagina aparecera como texto en negritas.

```HTML
<strong>Esta es mi primera pagina en HTML!!!</strong>
```

Si nosotros colocamos esto en nuestra pagina "index.html", la forma en la que podremos abrirlo es dando doble click en el archivo de la misma pagina, ya que es interpretada directamente por el navegador.
